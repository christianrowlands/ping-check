# Changelog

## [1.2](https://github.com/christianrowlands/ping-check/releases/tag/v1.2) - 2026-03-31

**IMPROVEMENTS**
* Remove the ACCESS_WIFI_STATE permission.

## [1.1](https://github.com/christianrowlands/ping-check/releases/tag/v1.1) - 2026-03-19

**BUG FIXES**
* Fix settings not syncing to the Ping tab.
* Fix numeric fields not being editable in settings.
* Fix status bar icons and latency chart theming.

## [1.0](https://github.com/christianrowlands/ping-check/releases/tag/v1.0) - 2026-03-17

**NEW FEATURES**
* Ping with configurable count, interval, packet size, and timeout.
* Latency charts with min, avg, max, loss, jitter, and stddev.
* Favorites with custom display names and per-host settings.
* Ping history with search.
* IPv4 and IPv6 support.
